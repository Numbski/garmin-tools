Upload Service

Introduction

The Upload Service allows you to perform uploads to the Garmin Connect API, and subsequently retrieve these imported files by user, activity or it's unique identifier.

As with the rest of our services, authentication and authorization are handled via OAuth, and is provided with a full SOAP and REST API.

REST

This API supports a Representational State Transfer (REST) model for accessing a set of resources through a fixed set of operations. The following resources are accessible through the RESTful model:

  • /fileAndReport
  • /report
  • /upload
  • /activity/{activityId}
  • /meta/{uploadId}
  • /upload/{uploadId}
  • /user/{username}
